Conventions

Clip loops the punch window at 0.25x.
Enter = capture the displayed frame, then ←/→ nudge by one frame, Enter/Space = confirm & next.
Esc = cancel a capture or the skip menu.
U = clear this punch's saved label.
S then 1/2/3 = skip (occluded / unclear / no punch in clip).
X = cycle speed 0.25 / 0.5 / 1x.

Impact = first frame the glove touches the bag. If there is no bag (shadowboxing), the first frame the fist stops moving toward its target. Not retraction start. Not maximum extension.
Shortcuts: Enter = capture / confirm  |  Space = play/pause (confirm once captured)  |  ←/→ = ±1 frame  |  Esc = cancel  |  U = clear label  |  S = skip  |  N = Next  |  P = Previous  |  F = First  |  G = Next unlabeled  |  X = speed
